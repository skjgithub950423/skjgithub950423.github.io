“||”只要有一边为true则为true，而”&&”只要有一边为false则为false,这是两边为布尔值的情况下，那么其他情况下呢，为此我查找了一些资料并进行了实验

逻辑运算符

“&&”(逻辑与)

如果val1 能转换成false则返回val1，否则返回val2。因此，与布尔值一起使用时，如果两个操作数都为true时&&返回true，否则返回false。

“||”(逻辑或)

如果val1能转换成true则返回val1，否则返回val2。因此，与布尔值一起使用时，如果任意一个操作数为true时||返回true。

能转换为false的表达式

  • null
  • NaN
  • 0
  • 空字符串（””）
  • undefined
